# Sirius vs Hammarby - Matchstatistik

Visualisering av matchstatistik från **IK Sirius - Hammarby IF 2-0**, Allsvenskan Omgång 2, 13 april 2026.

Data hämtad från [bolldata.se](https://bolldata.se/allsvenskan/matcher/2026/2026-04-13/sirius-hammarby-2-0).

## Funktioner

- Matchöversikt med slutresultat och xG
- Lagstatistik med visuella jämförelsestaplar
- Avslutskarta med xG-storlek och resultat
- Kumulativ xG-tidslinje
- Matchmomentum-graf
- Nyckelinsikter från matchen

## Teknikstack

- [Next.js 16](https://nextjs.org/) med App Router
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- SVG-baserade grafer (ingen extern diagrambibliotek)

## Komma igång

```bash
npm install
npm run dev
```

Öppna [http://localhost:3000](http://localhost:3000) i din webbläsare.
